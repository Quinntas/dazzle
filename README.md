# Go RestFull Template

This is a simple template for a RestFull API using Go.

## Getting Started

### Installing dependencies

```
go mod tidy
```

```
go mod vendor
```

### Building the project

```
go build -v ./...
```

### Running dev

```
air
```

### Running

```
go run ./...
```

### Running tests

```
go test -v ./...
```
